# 🏥 AI-Powered Medical Assistant for Disease Prediction and Healthcare Analytics

## 📌 Overview

This project is a Machine Learning-based healthcare application that predicts diseases based on user-entered symptoms. It also provides healthcare analytics, medical advice, prevention tips, and multilingual chatbot support using Streamlit.

---

## ✨ Features

- 🤖 AI Medical Chatbot (MediBot)
- 🩺 Disease Prediction using Machine Learning
- 📊 Healthcare Analytics Dashboard
- 🌍 English & Hindi Language Support
- 🔄 Automatic Symptom Translation
- 🏥 Department Recommendation
- 💊 Medical Advice
- 🛡 Prevention Tips
- 📈 Interactive Charts using Plotly
- 🎨 User-Friendly Streamlit Interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Deep Translator
- Machine Learning

---

## ▶️ How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
